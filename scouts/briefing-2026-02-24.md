### ✅ Critérios de Oportunidade

| Critério | Descrição | Peso |

| **MRR $10K-500K** | Revenue provável SaaS | Médio |

### 🎯 Categorias Prioritárias

1. **AI Code Generation** — Copilot alternatives, code completion especializado
2. **AI DevOps** — Deployment automation, infra as code inteligente
3. **AI Testing** — Test generation, bug prediction, QA automation
4. **AI Documentation** — Auto-doc, code explanation, API docs
5. **AI Debugging** — Error analysis, log intelligence, root cause

### ❌ O Que Evitar

- Wrappers simples da OpenAI API
- Ferramentas genéricas sem ângulo específico
- Ideias sem modelo de negócio claro
- Produtos que já têm 10+ competidores maduros
